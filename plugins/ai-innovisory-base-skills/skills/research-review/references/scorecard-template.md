## Notes on use

- **If the verdict is Ready**, the piece can be relied on as it stands, reasoning included. Recommended fixes are optional polish.
- **If the verdict is Use with caution**, the piece is usable for orientation but the user should address the recommended fixes before quoting it externally or using it to make a decision.
- **If the verdict is Draft only**, do not circulate or rely on the piece without rework. The blocking criterion (source quality, traceability, or argument quality) needs to be addressed first; the rest follows.
